## Getting Started
To build the place from scratch, use:

```bash
rojo build -o "VsCode adon.rbxlx"
```

Next, open `VsCode adon.rbxlx` in Roblox Studio and start the Rojo server:

```bash
rojo serve
```

For more help, check out [the Rojo documentation](https://rojo.space/docs).

## Gemini Prompt Setup
This project can generate prompts from Gemini via a server script in Roblox.

1. Create your local `.env` file (already included in this workspace) and set:

```bash
GEMINI_API_KEY=your_google_ai_studio_api_key_here
```

2. Generate `src/server/GeminiConfig.luau` from `.env`:

```powershell
./scripts/sync-env.ps1
```

3. In Roblox Studio, enable HTTP requests:
	- `Game Settings` -> `Security` -> enable `Allow HTTP Requests`

4. Run your game. The prompt panel requests prompts from Gemini through the server.

If Gemini fails or the key is missing, the game falls back to local random prompts.
